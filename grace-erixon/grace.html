Norris Coding Club Resources

Our class is over - here is how you can continue learning about web development and computer science!

Tutorial Resources

  • Codecademy Variety of programming languages, including web development.
  • Khan Academy Computer Programming Variety of programming languages, including web development.
  • Coursera Free with an Omaha library card. Variety of programming languages, including web development.
  • The Odin Project Focused on web development.
  • CodeWars Game-ified coding challenges that teach beginner to advanced.
  • Upskill Courses Mainly focused on web development.

Making Projects

  • Github This is what we used for this class. Create your own account at home.
  • Text Editor (like Notepad++) You can write websites in a normal text editor and run it locally in your browser.

Resources in Omaha

  • Omaha North STEM Summer Camps
  • Techademy
  • Once you reach middle school, there will be a lot more resources around Omaha. Make sure you reach out to adults at your school about your interest in computer science!
